# üí∏ Digital Payment Wallet (MERN Stack)

A secure and feature-rich **Digital Wallet Web Application** built using the **MERN Stack** that allows users to store funds, make electronic transactions, pay bills, manage wallets, and view dashboards with analytics.

---

##  Features

###  User Authentication
- Sign Up, Login (JWT based)
- Role-Based Access Control (Admin/User)
- Two-Factor Authentication (Email OTP)
- Password Reset via Email
- Email Verification

### Wallet Management
- Create/Manage Wallets
- View Wallet Balance
- Transaction History

###  Fund Management
- Add Money (UPI, Bank Transfer, Cards, PayPal)
- Transfer to Others (Phone/Email/Wallet ID)

###  Secure Transactions
- QR Code Payments
- Merchant Payments
- End-to-End Encryption

###  Dashboard & Analytics
- User Dashboard
- Admin Dashboard
- Graphical Reports

###  Bill Payments & Recharge
- Mobile Recharge
- Utility Bills
- Subscriptions
